# 🎬 Netflix Data Analysis Dashboard

This project is an interactive dashboard created using Power BI to analyze Netflix movies and TV shows.

## 📊 Features
- Total Shows, Movies, and TV Shows count (KPI cards)
- Genre analysis using bar charts
- Movies vs TV Shows comparison (Pie chart)
- Content trend over years (Line chart)
- Top countries producing content
- Interactive slicers for filtering data

## 🎨 Design
- Netflix-inspired dark theme
- Red highlight color for better visualization
- Clean and professional layout

## 🛠 Tools Used
- Power BI
- Data Visualization
